O Readme é um dos principais documentos de um projeto de softwere, é um arquivo de texto onde você irá descrever, 
documentar e exemplificar seu projeto.
No GitHub o Readme expõe seu projeto e é por ele que você poderá divulgá-lo, ou seja, o Readme é seu cartão de visitas.
O Readme possui extensão .md ou Markdown, que é uma linguagem de marcação usado para converter texto em código HTML, 
o markedown formata o texto usando apenas os símbolos do teclado.
